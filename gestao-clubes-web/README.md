# Gestão de Coletividades - Frontend

Aplicação React/Vite da plataforma de Gestão de Coletividades.

## Configuração

Defina `VITE_API_URL` com a origem do backend:

- Localhost: `http://localhost:8080`
- Online: URL pública do serviço no Render

No Vercel, configure `VITE_API_URL` nas variáveis de ambiente do projeto. Não coloque URLs de produção diretamente no código.
